Perplexity AI Tools, Extensions and Integrations: Complete 2026 List

PL
Prashant Lalwani 2026-04-22 · 12 min read
Perplexity AI AI Search

Perplexity AI is not just a website — it has a growing ecosystem of extensions, apps, and integrations. Here is the complete 2026 list of every tool, how it works, and whether it is worth adding to your workflow.

Try it free: Perplexity AI is at perplexity.ai — no account needed to start. Pro plan ($20/month) unlocks GPT-4o/Claude models, unlimited Pro Search, and file uploads.

1. Perplexity Web App (perplexity.ai)

The core product. Full-featured web interface with all Focus modes, conversational threads, file upload (Pro), and image generation (Pro). Works on any browser — no installation required. This is where most users spend most of their time.

Best for: Deep research sessions, conversational threads, academic and news research, file analysis.

2. Perplexity iOS App

Available free on the App Store. Includes voice input — speak your question and receive a synthesised answer read aloud. Supports all Focus modes, history sync with web account, and widget for quick queries from the home screen.

Best for: On-the-go research, voice queries while commuting, quick lookups away from desktop.

3. Perplexity Android App

Available on Google Play. Equivalent feature set to iOS. Supports Android voice assistant integration and Google Assistant replacement on some devices.

Best for: Mobile research, voice input, Android users wanting Google Search replacement.

4. Perplexity Chrome Extension

Install from the Chrome Web Store. Adds a Perplexity sidebar to any webpage — highlight text, right-click, and query Perplexity in context. Also enables setting Perplexity as the default search engine through the address bar.

How to install: Chrome Web Store → search "Perplexity AI" → Add to Chrome → Pin to toolbar.
Best for: In-context research while reading, setting as default search, quick lookups without leaving the current page.

5. Perplexity as Default Search Engine

Set up in any browser to replace Google in the address bar:

  • Chrome: Settings → Search engine → Manage → Add → URL: https://www.perplexity.ai/search?q=%s
  • Safari: Settings → Search → Search Engine → Other → enter Perplexity URL
  • Firefox: Add search engine from Settings → Search

After setup, type your question in the address bar and press Enter — Perplexity answers instead of Google.

6. Perplexity API (Sonar Models)

Perplexity offers its Sonar models via API for developers building applications:

ModelContextBest ForPrice/1M tokens
sonar128KFast search answers$1 in / $1 out
sonar-pro128KDeep research$3 in / $15 out
sonar-reasoning128KComplex reasoning$1 in / $5 out

The API is OpenAI-compatible — switching an existing OpenAI integration to Perplexity Sonar requires changing the base URL and model name only.

7. Perplexity in Raycast (Mac)

Raycast extension (free) lets you query Perplexity from the Raycast launcher without opening a browser. Type a keyboard shortcut → type your question → get an answer without leaving your current app. Excellent for power users who rely on Raycast.

8. Third-Party Integrations

Growing ecosystem of unofficial integrations:

  • Obsidian plugin — Query Perplexity directly from Obsidian notes
  • Notion integration — Bring Perplexity research into Notion databases
  • Zapier/Make — Connect Perplexity API to automation workflows
  • VS Code extension — Query Perplexity without leaving the editor

Recommended starting stack: Web app (primary) + Chrome Extension (set as default search) + iOS/Android app (mobile). This gives you Perplexity access everywhere with minimal setup. Add the API later when building custom workflows.
